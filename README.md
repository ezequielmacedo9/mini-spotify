# Mini Spotify em Java

Projeto desenvolvido em Java simulando funcionalidades básicas de um serviço de streaming de música.

## Funcionalidades
- Cadastro de músicas
- Criação de playlists
- Execução de músicas
- Menu interativo no terminal

## Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)
- Estruturas de Dados

## Como executar o projeto
1. Compile os arquivos `.java`
2. Execute a classe principal no terminal ou IDE

## Observações
Projeto desenvolvido para fins acadêmicos e prática de Java.
